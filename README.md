# AI Voice Assistant 🎙️🤖

An AI-powered voice assistant built using Python that can understand voice commands, respond naturally, and perform various tasks through AI and system automation.

## Features (Planned)

- 🎤 Speech-to-Text (Voice Recognition)
- 🗣️ Text-to-Speech Responses
- 💬 AI-powered Conversations
- 🖥️ Desktop Automation
- 🌐 Web Search
- 📅 Reminders and Notes
- 📄 PDF Summarization
- 🌦️ Weather Information
- 📊 System Monitoring
- 🔌 Extensible Plugin Support

## Tech Stack

- Python
- Flask / FastAPI
- HTML, CSS, JavaScript
- OpenAI API / Gemini API / Ollama
- SpeechRecognition / Whisper
- pyttsx3
- SQLite
