# 🚀 About EVI

EVI is not just another voice assistant.

It is an **AI-powered desktop productivity assistant** capable of understanding natural language, remembering user preferences, automating desktop tasks, answering questions using AI, interacting through voice and text, and assisting developers, students, and professionals.

Unlike traditional assistants that depend on fixed commands, **EVI understands intent** and can execute intelligent multi-step tasks.
